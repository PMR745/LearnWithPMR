Linear Search
What is Searching? Why do we need it?
Searching in Data Structures and Algorithm refers to process of finding a specific item or element in a given data structure such as an array or linked list. This can be done using various alogrithms such as linear search, binary search, and hash table search.
What is Linear Searching?
Linear Search method searches for an element by visiting all the elements sequentially until the element is found or data structure (eg. array) finishes.

Steps for Linear Search:-
  1. Set the first element of the array as the current element.
  2. If the current elment is the key, return its index.
  3. If the current element is not the key and if there are more elements in the array then increment the current element and repeat 2nd step.
  4. If the current element is not the key and there are no more elements in the array, return -1 to indicate the element was not found.
Linear Search Algorithm: 
int linearSearch (int arr[], int size, int key) {
    for(int i = 0; i < size; i++) {
        if(arr[i] == key) {
            return i;
        }
    }
    return -1;
}
Complexity:
Time Complexity = O(n)
Space Complexity = O(1)
