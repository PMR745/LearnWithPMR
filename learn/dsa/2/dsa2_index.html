Time Complexity
What is Time Complexity? Time Complexity is to study how effiecient an Algorithm is. And to anaylze how effiecient an Algorithm is, we need to know with which input the Algorithm takes less time and with which input it takes more time.

Now this anaylsis is done with following cases.

Best Case: Defines the input for which algorithm takes least time. Also known as lower bound. Denoted by Θ(theta) notation.

Average Case: Defines the input for which algorithm takes more than least time and less than maximum time. In between the lower and upper bound. Denoted by Ω(omega) notation.

Worst Case: Defines the input for which algorithm takes maximum time. Also known as upper bound. Denoted by O(Big O) notation.
Increasing order of common runtimes: 1 < log n < n < n log n < n2 < n2 log n < n3 < nn
