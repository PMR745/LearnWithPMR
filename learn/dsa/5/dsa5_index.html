Insertion Sort
Insertion sort is a sorting algortihm in which an unsorted element is placed at it's suitable positon in each iteraton.
Working of Insertion Sort:
  1. The first element in the array is assumed to be sorted so take the second element of the array and store it separately in current
    Compare current with the first element. If the first element is greater than current, then current is placed in front of the first element.
  2. Now, the first two elements are sorted.
    Take the third element and compare it with the elements on the left of it. Placed it just behind the element smaller than it. If there is no element smaller than it, then place it at the beginning of the array.
  3. Similarly, place every unsorted element at its correct position.
Insertion Sort Algorithm: 
for (int i = 1; i < n; i++)
{
    int current = arr[i];
    int j = i - 1;
    while (arr[j] > current && j >= 0)
    {
        arr[j+1] = arr[j];
        j--;
    }
    arr[j+1] = current;
    
}
