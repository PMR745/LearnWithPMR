Selection Sort
Selection sort algorithm sorts by repeatedly finding the minimum element from the unsorted array and putting it at the beginning.
Working of Insertion Sort:
  1. Set the first unsorted element as the minimum
  2. Traverse the array to find the minimum element in the array.
  3. While Traversing if any element is smaller than minimum then swap both the values.
  4. Then increment the minimum value to next element.
  5. Repeat until the array is sorted.
Insertion Sort Algorithm: 
for (int i = 0; i < n-1; i++)
{
    int min = i;
    for (int j = i + 1; j < n; j++)
    {
        if(arr[j]<arr[min]) {
            min = j;
        }
    }
    swap(arr[i], arr[min]);
}
Complexity:
Time Complexity = O(n2)
Space Complexity = O(1)
