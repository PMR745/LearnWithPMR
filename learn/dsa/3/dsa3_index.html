Sorting
Sorting refers to the process of arranging the sequence of given elements or data into any particular order, which may be ascending or descending. Sorting the data into a particular order makes it easier to use and perform operations for anaylsis.
Types of Sorting:
  • Bubble Sort
  • Selection Sort
  • Insertion Sort
  • Merge Sort
  • Quick Sort
  • Heap Sort
  • ...Many more
Why do we need so many Sorting Techniques?
Sorting algorithms get better day by day. For eg the radix sort which is the first ever sorting algorithm developed by American Computer genuis named Herman Hollerith.
Some of the Sorting techniques are much better than others in terms of time complexity. The above listed Sorting Techniques are important so we are learning them.
